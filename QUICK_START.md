# 🚀 Quick Start - NTOH Automação

## Começar em 3 passos

### 1️⃣ Instalar dependências
```bash
npm install
```

### 2️⃣ Iniciar servidor
```bash
npm run dev
```

### 3️⃣ Abrir no navegador
```
http://localhost:3000
```

---

## ✨ O que você pode fazer

### 📦 **Devoluções**
- Adicionar devoluções de clientes
- Rastrear status (Pendente → Processando → Concluído)
- Adicionar produtos devolvidos ao estoque

### 📊 **Estoque**
- Gerenciar inventário completo
- Visualizar valor total do estoque
- Adicionar/editar/remover produtos

### 💰 **Custos**
- Controlar custos fixos (aluguel, salários)
- Controlar custos variáveis
- Ver totalizações em tempo real

### 📈 **Faturamento**
- Visualizar gráficos de custos
- Análise de receita vs custos
- Resumo financeiro completo

---

## 💾 Dados

Todos os dados são salvos automaticamente no navegador (LocalStorage).
Nenhum servidor ou banco de dados é necessário!
